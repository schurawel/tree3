# Doing. Science. Theory.

Tree(3) (formerly ResearchGuide) is a research platform built around **TrigBiPy**, a novel theoretical approach to scientific methodology. This project is in active development and serves as a testbed for integrating human-machine collaboration in research workflows. And no, this is not an "AI" project. We are investigating different problems on the topic were Deep Learning could be a solution, but this is not the objective of this project. In fact this is basically the search for an Algorithm. The Algorithm of Thought itself. But we will explain that more detailed in future treatises.

## Overview

The goal is to develop a toolset that bridges rigorous theoretical principles with computational implementation, enabling researchers to structure and conduct investigations at the intersection of human insight and machine-assisted analysis.

**Note:** This is an early-stage prototype. A new Version is currently under development within a private repository. Currently it makes no sense to publish the other version because it is a major refactoring not usable by anyone. Minimum viable product stage will probably be reached end of 2026. Coherent papers and research materials will be provided earlier. The published version is still a proof of concept and not usable for actual research.

## Quick Start

```bash
make dependencies
# Build the application including documentation
make build-only
# Run raw Python application
make run
```
