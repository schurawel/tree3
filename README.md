# ResearchGuide

**Status: Early Prototype**

ResearchGuide is an experimental research framework built around **TrigBiPy**, a novel theoretical approach to scientific methodology. This project is in active development and serves as a testbed for integrating human-machine collaboration in research workflows.

## Overview

ResearchGuide explores the application of the TrigBiPy theoretical framework to practical research problems. The goal is to develop a platform that bridges rigorous theoretical principles with computational implementation, enabling researchers to structure and conduct investigations at the intersection of human insight and machine-assisted analysis.

**Note:** This is an early-stage prototype. APIs, workflows, and theoretical implementations are subject to change as the framework evolves.

## Quick Start

```bash
make dependencies
# Build the application including documentation
make build-only
# Run raw Python application
make run
```

## Project Structure

The codebase is primarily Python-based (93.4%), with shell scripting for build and deployment utilities (5.6%).

## Development

This project is actively being developed. The theoretical framework and implementation patterns are still being refined. Contributions, feedback, and discussions are welcome as we explore what works and what doesn't.
