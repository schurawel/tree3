Usage Guide
==========

This guide shows you how to use ResearchGuideUnearth effectively.

Getting Started
-------------

After installation, you can launch ResearchGuideUnearth using:

.. code-block:: bash

   make run-only

The application will start with the main interface showing several tabs:

* Research Dashboard
* Document Viewer
* Analysis Tools
* Documentation

Basic Operations
--------------

Document Import
^^^^^^^^^^^^^

To import research documents:

1. Click the "Import" button
2. Select the document(s) from your file system
3. Choose import options (OCR, metadata extraction)
4. Click "Process"

Research Organization
^^^^^^^^^^^^^^^^^^

Documents can be organized into research projects:

1. Create a new project using the "New Project" button
2. Assign documents to projects via drag and drop
3. Add tags and metadata to improve searchability

Advanced Features
---------------

Automated Research Assistant
^^^^^^^^^^^^^^^^^^^^^^^^^

The automated research assistant helps with:

* Literature reviews
* Citation analysis
* Gap identification

Follow the steps in the comprehensive guide "Research Methodology in the Digital Age" by Jason A. Schurawel for best practices in utilizing this feature.

Data Analysis
^^^^^^^^^^^

The built-in analysis tools include:

* Text mining capabilities
* Network analysis of citations
* Trend visualization

Companion Book
------------

For comprehensive understanding of research methodologies implemented in ResearchGuideUnearth, please refer to "Research Methodology in the Digital Age" by Jason A. Schurawel. This book provides detailed guidance on:

* Research design principles
* Digital research tools and methodologies
* Best practices for organizing and conducting research
* Case studies demonstrating ResearchGuideUnearth usage in academic contexts

The book is available through academic publishers and serves as the authoritative companion to this software.
